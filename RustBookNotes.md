## Ownership 
### Overview
Allows for error free memory management.

## Stack and heap
### Stack
Stack data structure.
Stack is fix sized.
Stores constants, function calls and pointers.

### Heap
Heap data structure.
Heap is variable sized.
Stores mutable data that will change durning runtime.
